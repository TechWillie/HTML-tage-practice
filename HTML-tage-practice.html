HTML Tags Cheat Sheet

Headings

Headings from `h1` to `h6` identify blocks and sections. Every page must have one `h1`.

Paragraphs

Paratgraphs of tect go in 'p' tags

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in 'ul' tags. Each list item goes in an 'il' tag.

Types of lists:

  • Unordered lists ('ul')
  • Ordered lists ('ol')

Ordered Lists

Ordered lists are numbered and go to 'oi' tags. each list itemn goes in an 'li' tag.

Anchors (i.e. links)

Anchor tags create links to other pages. the URL goes in the 'href' attribute. Go to Duckduckgo to search for more information.

Setting 'target="_blank" will open the link in a new window. See MDM for more details.

Anchor tags can slso create links to a new location on the same page. Instead of putting a URL goes in the 'href' attribute. Just use the 'id' of the element you want to navigate to. this allows you to jump to that part of the page.

Text Emphasis

Italic

the 'em element adds extra emphasis to the rtext within the opening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use the 'strongh' element to emphasize text. This shows up as bold tect in most browsers.
